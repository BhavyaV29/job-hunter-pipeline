Turnkey setup — no file editing

  1. Deploy the container (one-click on Render/Fly — see DEPLOY.md) or run locally: 
    pip install -r requirements-web.txt
uvicorn server:app --reload
  2. It comes up live and prints a one-time link with your admin token in the startup logs: 
    open http://localhost:8000/?token=…
    (or set your own ADMIN_TOKEN before starting).
  3. Open that link, go to settings, and add your API keys, connect a Google Sheet and pick your profile — all in the browser.
  4. Hit Run refresh on the dashboard. Optionally set ENABLE_SCHEDULER=1 for a daily auto-run.
  5. Hosting a public showcase for others? Deploy a second copy with DEMO_MODE=1 — it serves read-only sample data.
