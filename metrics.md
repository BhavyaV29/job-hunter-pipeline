# Weekly metrics - the feedback loop

Fill this in every Sunday (5 min). The point isn't vanity numbers - it's catching a
broken funnel early. If applications are high but responses are zero, the problem is
the resume or targeting, not effort. Adjust the next week based on what this shows.

## Diagnostic rules of thumb

- **Apps high, replies ~0%** -> resume/targeting problem. Get the resume reviewed; tighten target fit.
- **Replies but no tech screens** -> recruiter-screen story or resume-claim mismatch. Tighten STAR + headline.
- **Tech screens but no onsites** -> DSA/communication gap. More AI-free reps + mock interviews.
- **Referrals converting >> cold apps** (almost always true) -> shift even more time to outreach.
- **Posts driving profile views / DMs** -> double down on the pillar that's landing.
